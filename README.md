# 🌌 AI Multiverse – Memory Vault Chatbot

## Description

AI Multiverse is a Streamlit-based chatbot powered by the Google Gemini API. Users can interact with different cricket personalities through a modern chat interface. The application uses Streamlit Session State (`st.session_state`) to remember conversations, allowing chat history to persist even after app reruns or sidebar changes.

## Features

* 🏏 Chat with multiple cricket personalities
* 🤖 Powered by Google Gemini API
* 💬 Modern chat interface using `st.chat_input` and `st.chat_message`
* 🧠 Persistent conversation history using `st.session_state`
* 🎚️ Adjustable personality intensity slider
* 📱 Simple and responsive Streamlit interface

## Technologies Used

* Python
* Streamlit
* Google Gemini API (`google-genai`)
* python-dotenv
